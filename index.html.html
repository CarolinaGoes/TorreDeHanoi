Torre de Hanói

Este é um jogo clássico de matemática e lógica.

O objetivo é mover todos os discos de uma haste para outra, seguindo certas regras.

Regras do jogo

  • Somente um disco pode ser movido por vez.
  • Não é permitido colocar um disco maior sobre um menor.
  • Apenas o disco do topo de uma haste pode ser movido.
